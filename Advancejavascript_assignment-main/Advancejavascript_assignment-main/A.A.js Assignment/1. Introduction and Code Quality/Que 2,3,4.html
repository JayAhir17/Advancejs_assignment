4. What is the code below going to output? Alert( null || 2 || undefined );

ANS:- This code will output 2. In JavaScript, the logical OR (||) operator returns the first truthy operand or the last operand if all operands are falsy. In this case, null is falsy, 2 is truthy, so the expression null || 2 evaluates to 2. Therefore, the alert will output 2.
